# LegacyChain

A decentralized application (dApp) developed during the **Sui Romanian Hackathon**. LegacyChain provides a blockchain-based solution for digital inheritance and supply chain transparency, focusing on the secure integration of legacy data into modern distributed ledgers.

## Technical Features
- **Full-Stack Development:** Designed and implemented both the Frontend interface and the Backend logic to interact with the blockchain.
- **Sui Blockchain Integration:** Real-time interaction with Move-based smart contracts for data verification and asset management.
- **Decentralized Storage:** Ensuring data integrity and transparency across the supply chain.
- **Modern UI/UX:** Built a responsive dashboard for tracking legacy assets and verifying their origin.

## Built With
- **Blockchain:** Sui Network (Move Language)
- **Frontend:** React.js / TypeScript
- **Backend:** Node.js
- **Tools:** Sui Wallet, SDK integration, NPM

## Project Structure
- `dapp/` - The React-based frontend interface and client-side logic.
- `smart_contract/` - Move modules, smart contract logic, and blockchain deployment files.
- `package.json` - Project dependencies and configuration for the development environment.
- `.hintrc` - Linting configuration for maintaining code quality.
